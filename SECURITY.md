# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in Ritma, please report it privately.

- Prefer GitHub: use **Security Advisories** for this repository (Security tab â†’ "Report a vulnerability").
- If GitHub Security Advisories are not available in your environment, open a GitHub issue with **no sensitive details** and request a private contact channel.

Please include:
- A clear description of the issue and impact
- Reproduction steps (as minimal as possible)
- Affected versions/commits (if known)
- Any potential mitigations

## Supported Versions

Security fixes are provided for the latest released version.

## Disclosure

We aim to acknowledge reports promptly and coordinate a responsible disclosure timeline.
